HTML Basics

HTML stands for HyperText Markup Language.

"Hypertext" is text that contains links to other texts, and this includes hypermedia. "Markup" refers to the annotative nature of the tags and elements used in HTML. "Language" implies syntax and semantics or, in other words, that there is a particular grammar to this language and a particular meaning to tags which communicate structure to machines and humans.

The History of HTML

Pre 1997: "Wild Wild Web"
1997: HTML 4.0 & 4.01 became standardized
2000: XHTML 1.0
2004: WHATWG
2007: W3C conceded and began to work with WHATWG

W3C standardized HTML5
WHATWG: Web Hypertext Application Technology Group. A group of browser vendors toward HTML that believes HTML is constantly evolving with no ascription to a "standard."

div and span

div (characterized by its tags <div></div>) and span (<span></span>) are taught together - I am still figuring out why.
div is the most generic 'block element,' which renders content to begin on a new line by default and may contain inline or other block-level elements. Block element can be roughly translated to 'flow content.'
span is an 'inline element,' which renders content on the same line by default. Inline elements may only contain other inline elements, and can roughly be translated to 'phrasing content.'
